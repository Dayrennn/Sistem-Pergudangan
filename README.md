# Sistem Informasi Pergudangan Berbasis Web

Sistem Informasi Pergudangan Berbasis Web adalah aplikasi yang dikembangkan untuk membantu pengelolaan stok, distribusi, dan pencatatan data pergudangan secara terstruktur dan terintegrasi. 
Project ini dibuat sebagai media pembelajaran sekaligus portofolio, dengan studi kasus optimalisasi stok dan alur distribusi barang pada perusahaan.

Project Purpose
1. Tujuan pengembangan sistem ini adalah:
2. Membangun sistem pergudangan berbasis web yang terstruktur
3. Mengoptimalkan pengelolaan stok barang masuk dan keluar
4. Mengurangi kesalahan pencatatan data secara manual
5. Menerapkan konsep CRUD dan relasi database
6. Mengimplementasikan role-based access (admin, marketing, direktur, dll.)
7. Meningkatkan pemahaman pengembangan sistem informasi berbasis web

Features
1. Fitur utama yang terdapat pada sistem ini meliputi:
2. Manajemen data barang
3. Pencatatan barang masuk dan barang keluar
4. Manajemen produk jadi dan produk terjual
5. Manajemen supplier dan pelanggan
6. Sistem penjualan dan pemesanan
7. Update stok otomatis berdasarkan transaksi
8. Sistem usulan dan permintaan pembelian
9. Validasi stok dan perhitungan harga
10. Hak akses pengguna berdasarkan role

Tech Stack
1. PHP
2. MySQL
3. HTML, CSS, JavaScript
4. Bootstrap
5. Web-based CRUD System

Project Scope
Sistem ini dikembangkan untuk mendukung aktivitas pergudangan seperti:
1. Monitoring ketersediaan stok secara real-time
2. Pencatatan transaksi barang dan penjualan
3. Pengelolaan data supplier dan pelanggan
4. Penyusunan laporan data pergudangan

Project Status
1. Project ini telah selesai bisa di cek melalui trimitraabadilestari.com.
2. Fitur akan terus dikembangkan sesuai kebutuhan sistem dan peningkatan kualitas aplikasi.

Notes
1. Project ini dibuat untuk tujuan pembelajaran dan akademik
2. Tidak ditujukan untuk penggunaan production
3. Struktur sistem dapat berubah sesuai kebutuhan pengembangan
